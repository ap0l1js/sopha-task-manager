## 💚 Sopha Task Manager

This project is a fullStack task manager.

## ⚙️ Technologies

- React (Nextjs)
- TypeScript
- Shadcn UI
- Tailwind
- Unsplash
- Clerk
- Dnd

### Features in this application.

- 🤯 Multiboard
- 🏢 Favorite boards
- 🌐 Drag and Drop
- 🚀 Assign users to task
- 💻 Invitations
- 🔄 Multiple Organizations
- 🔐 Security Clerk authentication
- 📊 Logs
- 📌 Google social login
- 🎨 Customized board picture
- 🔗 Notifications
